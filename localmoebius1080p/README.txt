This .zip file contains:

% localmoebius.m

% crossmoebius.m

% qinterp2.m

% dispvectors.m

% gaussian2d.m

% localmoebius_interface.m

% localmoebius_interface.fig

% rodela.jpg (thanks to Luca Biada from Flickr)

---------------------------------------------------------------------

Using the interface:

% Unpack this file in a folder

% Open your MATLAB version

% set the command guide in MATLAB environment

% Open the localmoebius_interface.fig

% Change to folder (recommended***) or just add to path

% Run the interface

*** We strong recommend it because the .dat files generated in

 "Save Results" button will be saved in your current folder.
